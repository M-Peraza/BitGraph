Example of project that links with BitScanPlus repo
Includes a CMakeLists.txt and a main.cpp source file

I. Deploy BitScanPlus using CMake install command. 
Set INSTALL_PATH variable as ROOT directory (CMake-GUI). By default it is this folder. This will create appropiate folders lib and include.

II. Configure this target with the given CMakeLists.txt file as usual
